Welcome to MattCraft

A modded Forge 1.16.5 server

MattCraft is a modded Minecraft Forge server focused on increasing the difficulty and complexity of the game in a steampunk era.

The server can be found at:

  • 40.112.128.105
  • matt-server.westus.cloudapp.azure.com

A download link for the mod pack can be found on the installation page.


Installation instructions can be found alongside the mod pack on the installation page. If you know what you're doing you can also download it manually here.
