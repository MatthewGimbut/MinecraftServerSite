How to install the mod pack

  • 0. Download and install Java 17 if it or a newer version isn't already installed on your computer.
  • 1. Download the CurseForge client.
  • 2. Navigate to the MattCraft modpack.
  • 3. Click files and select the most recent version.
  • 4. Click "Install" and if prompted, "Open link". This will open the modpack in the CurseForge client and begin downloading the mods.
  • 5. Once complete, click the modpack and then click play. You're now ready to join the server!

If you're experiencing crashes while playing, try clicking the modpack in the CurseForge client, the three dots next to "Play" for settings, and uncheck "Use System Memory Settings". Drag the slider so that it's 8000-9000MB.

If you can't see Immersive Engineering components, make sure to set "enableVBO" in the IE config file to false.


How to configure your game [optional]

With over 150 mods, there are a lot of overlapping keybindings. These can all be managed in your control settings and are sorted by mod name. Here are a few suggestions to get you started...

  • Sophisticated Backpacks - Bind "Open Backpack".
  • Voice Chat - Bind "Voice Chat Settings" and "Push To Talk".

How to install shaders [optional]

Optifine is the popular choice but it has many compatibility issues and will likely lead to crashes with the modpack. We recommend the following if you're interested in shaders...

  • 0. Download the Oculus and Embeddium .jar files for Minecraft version 1.20.1.
  • 1. Download the shader pack of your choice.
  • 2. In the CurseForge client, select the modpack and click "Open Folder" in the options menu next to "Play".
  • 3. Put the two .jar files from step 0 in the "mods" folder. These mods are clientside only and will not affect the modpack installation.
  • 4. Put your shaderpack .zip in the "shaderpacks" folder.

You can now launch the game and configure your shaders in the "Video Settings..." option.
