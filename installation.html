How to install the mod pack

  • Download the server mod pack here.
  • Download the Minecraft Forge version 1.16.5.-36.2.34 installer. You will need to open the 1.16 dropdown on the left and then click show all versions to see it.
  • If prompted for confirmation when opening the file, click "OK".
  • Make sure "Install client" is selected and the file location is your Minecraft folder and hit "OK".
  • Open the Minecraft launcher, click Installations, then the "..." next to the Forge entry, and "Edit".
  • Click the dropdown for more options.
  • Paste the following into the "JVM ARGUMENTS" textbox and hit "Save": -XX:+UseG1GC -Xmx7G -Xms4G -Dsun.rmi.dgc.server.gcInterval=2147483646 -XX:+UnlockExperimentalVMOptions -XX:G1NewSizePercent=20 -XX:G1ReservePercent=20 -XX:MaxGCPauseMillis=50 -XX:G1HeapRegionSize=32M -XX:ParallelGCThreads=4
  • From the Minecraft launcher, press "Play" with the new Forge installation selected.
  • From the main menu, click Mods, and then Open Mods Folder.
  • Copy the .jar files in this zip from step 1 to that folder.
  • Restart Minecraft.

You should be able to connect to the server after completing these steps.

If you can't see Immersive Engineering components, make sure to set "enableVBO" in the IE config file to false.
