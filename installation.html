How to install the mod pack

  • Download and install the client version of Minecraft Forge version 1.15.2.-31.2.44.
  • Download the server mod pack here.
  • From the Minecraft launcher, click Installations and then the "..." next to the Forge entry.
  • Click the dropdown for more options.
  • In the JVM Arguments line, change -Xmx2G to something higher to allow Forge to use more RAM. I suggest 4G or 5G.
  • From the Minecraft launcher, select the new version of Forge and launch it.
  • From the main menu, click Mods, and then Open Mods Folder.
  • Extract the .jar files in this zip to that folder and launch Minecraft again.

You should be able to connect to the server after completing these steps.

If you get errors about needing to download MixinBootstrap, you can download the mod here.

If you can't see Immersive Engineering components, make sure to set "enableVBO" in the IE config file to false.
