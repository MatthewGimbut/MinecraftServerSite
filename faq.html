Why does my game crash when launching Forge?
You probably just need to allocate more RAM to the JVM. Go into the Forge launch options and change -Xmx8G to something higher. This increases the max memory that Minecraft can use.
Can I suggest a mod be added?
Sure, contact me either through the site or on Discord if you'd like to see something added.
What is the server player limit?
Currently, the server is set to a max of 32 concurrent players.
Do you accept donations?
We do not currently accept donations.
Where is the server located?
The server is based in Seattle. You should be able to connect from the east coast with no issues.
