AttributeFix

Minecraft uses an attribute system to handle important calculations. It covers everything from armor values, to attack damage, to the maximum health of a mob. This system also introduced attribute caps, which limit how high any value can be. This mod significantly increases these caps to allow mods to work as they were intended.
